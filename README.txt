# Employee Directory
By: Nate Jaeger

## Functionality
Fetch 12 users using the RandomUser API
	-Creates div cards to display user information
	-Creates Modals to display more user info upon card click
	-Can scroll though employee Modal windows using 'next' and 'prev' buttons
	-Can search for and display found employees from shown list

### Style Changes

-Added box shadow to card elements
-Bolded email and location fonts on cards
-Changed border radius of pictures on cards
-Made Modal name larger
